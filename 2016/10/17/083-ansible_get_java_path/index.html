发表于 2016-10-17   |   分类于 devops   |  

问题背景

最近公司要做运维自动化(工具使用ansible)，但因为前期未统一环境，导致线上机器JAVA_HOME并不统一
于是有了一个需求，如何使用ansible去获取JAVA_HOME？
