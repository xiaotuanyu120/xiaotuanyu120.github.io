什么是easysnmp？

  easysnmp是官方Net-SNMP Python Bindings的一个分支，致力于让net-snmp更python。

easysnmp的特点

  • 传统的net-snmp很优秀，但是太不python和缺少合适的unit去测试和文档化；
  • PySNMP是用纯净的python创造的，然而它有很大的性能问题。easysnmp大概是其的4倍速度；
