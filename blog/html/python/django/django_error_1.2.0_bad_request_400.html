django error: 1.2.0 "bad request 400"

1. 错误信息

当设置DEBUG为False时，访问页面出现bad request 400错误

2. 解决办法

1) 原因

没有在settings.py中配置ALLOWED_HOSTS

2) 简单粗暴的解决方法就是配置成所有主机可访问:

DEBUG = False

ALLOWED_HOSTS = ['*']

详细配置参看： django documents
