views: 2.0 获取用户ip

15 Nov 2016

1. 在views中获取用户ip

views.py

def user_ip(request):
    try:
        return request.META.get('HTTP_X_FORWARDED_FOR').split(',')[0]
    except:
        return request.META.get('REMOTE_ADDR')
